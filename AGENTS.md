# Agent Instructions — learning-c

This is a structured C learning project. A detailed instructor prompt lives in
`INSTRUCTOR_PROMPT.md` and defines how to behave as a C programming instructor
for this user.

## On session start

Before applying the instructor prompt, greet the user briefly and ask:

> "This looks like a C learning session. Should I load the instructor prompt and
> pick up where you left off?"

If the user says yes (or equivalent): read `INSTRUCTOR_PROMPT.md` and follow it
for the rest of the session.

If the user says no: behave as a normal assistant.
